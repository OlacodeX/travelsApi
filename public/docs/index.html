Introduction

This is the docs for travel and tours API

Base URL: http://localhost

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

To authenticate requests, include an Authorization header with the value "Bearer {YOUR_AUTH_KEY}".

All authenticated endpoints are marked with a requires authentication badge in the documentation below.

You can retrieve your token by visiting your dashboard and clicking Generate API token.

Admin

Add a new travel. This endpoint allows an admin to create a new travel

requires authentication

Example request:
curl --request POST \
    "http://localhost/api/v1/admin/travels" \
    --header "Authorization: Bearer {YOUR_AUTH_KEY}" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"is_public\": true,
    \"name\": \"My Travel\",
    \"description\": \"Et deserunt sint impedit nihil expedita voluptates sunt.\",
    \"number_of_days\": 20
}"

const url = new URL(
    "http://localhost/api/v1/admin/travels"
);

const headers = {
    "Authorization": "Bearer {YOUR_AUTH_KEY}",
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "is_public": true,
    "name": "My Travel",
    "description": "Et deserunt sint impedit nihil expedita voluptates sunt.",
    "number_of_days": 20
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());
$client = new \GuzzleHttp\Client();
$response = $client->post(
    'http://localhost/api/v1/admin/travels',
    [
        'headers' => [
            'Authorization' => 'Bearer {YOUR_AUTH_KEY}',
            'Content-Type' => 'application/json',
            'Accept' => 'application/json',
        ],
        'json' => [
            'is_public' => true,
            'name' => 'My Travel',
            'description' => 'Et deserunt sint impedit nihil expedita voluptates sunt.',
            'number_of_days' => 20,
        ],
    ]
);
$body = $response->getBody();
print_r(json_decode((string) $body));

List of Tours. This endpoint return all tours for a particular travel You can also filter the result with price, starting date and ending date

requires authentication

Example request:
curl --request GET \
    --get "http://localhost/api/v1/travels/laborum/tours?sortBy=eos&sortOrder=dolor" \
    --header "Authorization: Bearer {YOUR_AUTH_KEY}" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"priceFrom\": 2.910355351,
    \"priceTo\": 18567352.29,
    \"dateTo\": \"2023-07-03T09:25:18\",
    \"dateFrom\": \"2023-07-03T09:25:18\",
    \"sortBy\": \"price\",
    \"sortOrder\": \"desc\"
}"

const url = new URL(
    "http://localhost/api/v1/travels/laborum/tours"
);

const params = {
    "sortBy": "eos",
    "sortOrder": "dolor",
};
Object.keys(params)
    .forEach(key => url.searchParams.append(key, params[key]));

const headers = {
    "Authorization": "Bearer {YOUR_AUTH_KEY}",
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "priceFrom": 2.910355351,
    "priceTo": 18567352.29,
    "dateTo": "2023-07-03T09:25:18",
    "dateFrom": "2023-07-03T09:25:18",
    "sortBy": "price",
    "sortOrder": "desc"
};

fetch(url, {
    method: "GET",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());
$client = new \GuzzleHttp\Client();
$response = $client->get(
    'http://localhost/api/v1/travels/laborum/tours',
    [
        'headers' => [
            'Authorization' => 'Bearer {YOUR_AUTH_KEY}',
            'Content-Type' => 'application/json',
            'Accept' => 'application/json',
        ],
        'query' => [
            'sortBy' => 'eos',
            'sortOrder' => 'dolor',
        ],
        'json' => [
            'priceFrom' => 2.910355351,
            'priceTo' => 18567352.29,
            'dateTo' => '2023-07-03T09:25:18',
            'dateFrom' => '2023-07-03T09:25:18',
            'sortBy' => 'price',
            'sortOrder' => 'desc',
        ],
    ]
);
$body = $response->getBody();
print_r(json_decode((string) $body));

Request      

GET api/v1/travels/{travel_slug}/tours

Headers

Authorization      

Example: Bearer {YOUR_AUTH_KEY}

Content-Type      

Example: application/json

Accept      

Example: application/json

URL Parameters

travel_slug   string   

The slug of the travel. Example: laborum

Query Parameters

sortBy   string  optional  

Field to sort by. Accepts only price as value Example: eos

sortOrder   string  optional  

Order of sorting. Accepts either asc or desc as value Example: dolor

Body Parameters

priceFrom   number  optional  

Example: 2.910355351

priceTo   number  optional  

Example: 18567352.29

dateTo   string  optional  

Must be a valid date. Example: 2023-07-03T09:25:18

dateFrom   string  optional  

Must be a valid date. Example: 2023-07-03T09:25:18

sortBy   string  optional  

Must be one of price. Example: price

sortOrder   string  optional  

Must be one of asc or desc. Example: desc
